# English Wh-words Test

A comprehensive online test for assessing students' understanding of Wh-words in English.

## Features

- ✅ 25 multiple-choice questions
- ✅ 10-minute timer with auto-submission
- ✅ Page leave detection (max 3 leaves)
- ✅ All questions on one page
- ✅ Real-time progress tracking
- ✅ Telegram notifications
- ✅ Mobile-responsive design
- ✅ Detailed student reports

## Deployment

### 1. Setup Telegram Bot (Optional)

1. Message `@BotFather` on Telegram
2. Create a new bot with `/newbot`
3. Get your bot token
4. Start a chat with your bot and send a message
5. Visit `https://api.telegram.org/bot<YourBOTToken>/getUpdates` to get your chat ID

### 2. Deploy to Vercel

1. Create a new GitHub repository with these files
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Add environment variables in Vercel dashboard:
   - `TELEGRAM_BOT_TOKEN`: Your Telegram bot token
   - `TELEGRAM_CHAT_ID`: Your Telegram chat ID
5. Deploy!
